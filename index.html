🩸 AnemiaAI Pro

Non-Invasive Anemia Detection System

This project uses Deep Learning models like DenseNet121, VGG16, and InceptionV3 to detect anemia from conjunctiva eye images with high accuracy.

🤖 AI Models

DenseNet121, VGG16, InceptionV3 with ensemble learning.

📊 Accuracy

Achieved up to 100% accuracy using ensemble approach.

💻 GUI App

Interactive desktop app using Tkinter for easy usage.

⚡ CLI Tool

Fast prediction using command line interface.

🔄 CI/CD

Automated pipeline using GitHub Actions.

🌐 Deployment

Live website hosted using GitHub Pages.
