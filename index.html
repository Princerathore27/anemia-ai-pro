🩸 AnemiaAI Pro

Non-Invasive Anemia Detection System

This project uses Deep Learning models like DenseNet121, VGG16, and InceptionV3 to detect anemia from eye images.

Features:

  • ✔️ Deep Learning Models
  • ✔️ Ensemble Prediction
  • ✔️ Desktop GUI App
  • ✔️ CLI Tool
  • ✔️ CI/CD Integrated
