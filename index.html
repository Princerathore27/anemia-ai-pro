🩸 AnemiaAI Pro

AI-Based Non-Invasive Anemia Detection

Deep learning based system using DenseNet121, VGG16 & InceptionV3 to detect anemia from eye images.

🚀 Features

🤖 AI Models

DenseNet121, VGG16, InceptionV3 Ensemble

📊 Accuracy

Up to 100% accuracy achieved

💻 GUI

Interactive Tkinter Desktop App

⚡ CLI

Fast command-line predictions

🔄 CI/CD

Automated using GitHub Actions

🌐 Deployment

Live on GitHub Pages

1. Upload eye image → 2. Model processes → 3. Prediction shown (Anemic / Non-Anemic)
